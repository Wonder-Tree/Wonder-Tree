Human Pose Estimation SDK

Our SDK (Software Development Kit) named as `PoseCamera` is a python package which provides fast Human Pose Estimation using normal RGB camera. Its optimized for low edge devices means you can run it on cpu device easily unlike other Human Pose Estimation deep learning solutions which requires heavy GPU's to run the software. You can build pretty simple games which helps the inclusion community and become the part of our community.
